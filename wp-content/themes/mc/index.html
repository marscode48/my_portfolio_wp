SKILL

HTML5

CanvasやSVGなどのグラフィックスを得意とし、またSEOの最適化やメンテナンス性を保ったセマンティックなマークアップを提供します。

CSS3

BEMを使ったCSS設計により、アニメーションやグラデーションを取り入れた高度なCSSを保守性の高い、拡張性のあるCSSコードで作成します。

SASS

LibSassからDartSassに対応し、CSSの記述をより効率的かつ柔軟に行い、保守や管理のしやすいスタイルシートを提供します。

JavaScript

オブジェクト指向によるclassを取り入れたコーディングにより、保守性、再利用性、拡張性を担保したJavaScriptを提供します。

GSAP

ウェブアニメーションの制作には欠かせないJavaScriptライブラリであるGSAPを得意とし、高性能で滑らかなアニメーションを作成します。

Three.js

3DグラフィックスのJavaScriptライブラリであるThree.jsを使い、魅力的な3Dコンテンツが体験できるWebサイトを提供します。

Vue.js

Vueを使ってユーザーインターフェースを構築し、心が惹かれるウェブアプリケーションやSPAを開発し、ユーザーに魅力的な体験を提供します。

Git, GitHub

Gitを使ったバージョン管理ツールを行い、チームとの協力やバージョン管理において、円滑な連携や作業を実現し、スピーディーな開発を実行します。

Shopify

Liquidを使用して、オンラインストアのテーマやページのカスタマイズを行い、事業主様が理想とするデザインのオンラインストアを構築します。

WordPress

オリジナルテーマの作成はもちろん、デザインや機能を自由にカスタマイズしたカスタム投稿や分類などを、管理者の使いやすさと合わせて提供します。

ABOUT

  • #01

    Welcome!

    はじめまして、神奈川県を拠点に全国で活動するWeb制作フリーランスのコーダー、足立 雅和（アダチ マサカズ）と申します。

    以前の職場で知人に制作したホームページが予想以上に好評だったことが、私がWeb制作という新たな道を選ぶきっかけとなりました。

    お客様の喜ぶ笑顔をより多く見るため、そしてその喜びをもっと分かち合いたいという想いから、現在はWeb制作フリーランスとして活動しています。

  • #02

    Quality Coding

    コーダーとして最も大事なこと。それは、デザイナーから提供されたデザインカンプを忠実に再現することだと考えています。

    私はピクセルパーフェクトな実装を追求し、さまざまなブラウザでの動作を確認して表示の一貫性を確保します。

    また、他のエンジニアがコードを改善することを前提に、拡張性と保守性の高いコードを書くことで、常にコーディングの品質を高めるように努めています。

  • #03

    Communication

    適切なコミュニケーションは、フリーランスとしてご依頼主様との信頼を築く上で不可欠な要素です。

    私は積極的に進歩を報告し、頻繁に連絡を取り、必要に応じて相談することで、強固な信頼関係を築くことを常に心がけています。

    また、案件対応中に不明点が生じた場合は、独断的な判断を避け、ご依頼主様のご要望を丁寧に再確認します。これにより、誤解を防ぎ、円滑な作業の進行を実現します。

  • #04

    Responsibility

    ご依頼頂いた案件につきましては、ご期待に応えられるよう、常に責任感を持ち、誠心誠意取り組みます。

    また、時間管理には特に細心の注意を払い、タスクや案件の納期を厳守し、ご依頼主様の計画通りにプロジェクトを円滑に進めます。

    万が一、緊急の修正や問題が発生した場合でも、迅速かつ適切に対処し、ご依頼主様のご要望に最善の形でお応えいたします。
